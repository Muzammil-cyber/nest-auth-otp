## Description

Secure NestJS API with:

- JWT authentication and refresh tokens
- OTP email verification via `@nestjs-modules/mailer`
- Role-based authorization (RBAC)
- MongoDB (Mongoose)
- Swagger docs at `/api/docs`

## Project setup

```bash
pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# e2e tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
